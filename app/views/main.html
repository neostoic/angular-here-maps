Angular Here Maps

AngularJS directives to integrate Nokia Here Maps into your applications - An alternative to Google Maps

Download (1.0.6) View on GitHub

About

Angular Here Maps is a set of directives written in Javascript which integrates Nokia Here Maps in an AngularJS application.

It is based on the Nokia Here Maps Javascript API version 4.

Features

There are directives for some of the Here Maps objects, including markers and windows. We're currently working on integrating more layer objects, including lines and shapes and such.

Contributing

The code is hosted on GitHub. If you need a feature which isn't implemented yet, please use our issue tracker to open a feature request, or fork our repository, implement the feature and issue a pull request. Contributions are more than welcome!

Branching model

We chose git flow as our branching model. The main development is done in the develop branch. The master branch almost always reflects the latest tagged version, but sometimes some quick fixes can land into it.

If you are interested in contributing, we encourage you to learn about git flow.
