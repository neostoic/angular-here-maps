Getting started

  1. Download angular-here-maps.js (minified version) and put it with your other scripts.

    you can also use Bower to install it automatically

    bower install angular-here-maps --save

  2. Load angular-here-maps.js into your HTML page:

    <script src="/path/to/bower_components/angular-here-maps/build/angular-here-maps.min.js" type="text/javascript"></script>

  3. Include the Here Maps API v3 Javascript files:

    <script src="/path/to/bower_components/here-maps-api/modules/mapsjs-core.js"></script>
<script src="/path/to/bower_components/here-maps-api/modules/service/mapsjs-service.js">

  4. Make your application module depend on the angular-here-maps module:

    var app = angular.module('myApp', ['angular-here-maps']);

  5. Configure Angular Here Maps:

    .config(function(MapConfigProvider) {
    MapConfigProvider.setOptions({
        appId: 'your Here Maps app id',
        appCode: 'your Here Maps app code'
    });
})

    You will need an appId and appCode which you can get here:

  6. Add a map object to your scope with your zoom and center values:

    $scope.map = {
  zoom: 14,
  center: { 
    lng: -0.135559,
    lat: 51.513872
  }
}

  7. include the map element in your html:

    <map zoom="map.zoom" center="map.center"></map>

  8. Specify a height for the here map container:

    .angular-google-map-container { height: 400px; }
